# 동적 DCA 백테스트 & 포트폴리오 도구

기계식(정액) DCA와 **동적 DCA**(이동평균 이격도 + 전고점 낙폭 신호로 매수 배수 조절, 남는 현금은 SGOV/단기국채 이자 운용)를 비교하는 인터랙티브 웹앱입니다. Yahoo Finance 데이터를 사용하며 Vercel 서버리스 함수로 임의 티커를 조회합니다.

## 주요 기능

- **기계식 vs 동적 DCA 백테스트** — 자산가치·매수배수·달러효율 그래프, IRR/MDD 지표
- **예산 모델** — `총원금 동일(리저브)` / `하락 시 추가금(IRR 비교)`
- **월납 / 주납** 적립 주기, 시작~종료 기간 지정
- **임의 티커 + 포트폴리오** — 여러 종목 비중 합산, 자동 리밸런싱(적립금 드리프트 / 주기적 절삭 / 저평가 이동)
- **실행 플랜** — 적립금 → SGOV 예치/인출 → 종목별 매일 매수액
- **내 현재 포트폴리오** — 보유·평단 입력 → 도넛 차트 + 목표 비중 조정안
- **배당 & 재투자** — 종목별 지급주기·배당률, 배당 전용 투자처 지정
- **신규 진입 3년 전망** — 역사적 3년 DCA 분포 기반 상승/평균/하락 시나리오

## 로컬 실행

```bash
# 정적 파일만 (사전 저장된 QQQ·VOO 동작)
# index.html 을 브라우저로 열기

# 임의 티커까지 (서버리스 함수 포함)
npm i -g vercel
vercel dev            # http://localhost:3000
```

## 데이터 갱신

```bash
node build_data.js    # QQQ·VOO·^IRX·SGOV 가격/배당 → data.js, CSV 재생성
```

## 배포 (Vercel)

이 저장소를 Vercel에 임포트하면 자동 배포됩니다. `api/quote.js`가 서버리스 함수로 동작해 CORS 없이 Yahoo Finance를 프록시합니다.

## 구조

| 파일 | 설명 |
|------|------|
| `index.html`, `styles.css`, `app.js` | 프론트엔드 + 백테스트 엔진 |
| `api/quote.js` | Vercel 서버리스 (Yahoo 프록시, 가격+배당) |
| `build_data.js` | 오프라인 폴백용 `data.js` 생성 |
| `data.js` | 사전 저장 데이터(QQQ·VOO·^IRX·SGOV) |
| `chart.umd.min.js` | Chart.js (로컬 번들) |

## 면책
